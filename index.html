Reel Developer

Become a better Java developer or prepare for developer interviews by creating apps inspired by trailers.

Choose a Process

Option 1: Netflix Trailers

  1. Go to Netflix’s YouTube channel: https://www.youtube.com/c/Netflix/videos
  2. Select a random video trailer
  3. Watch the trailer
  4. Make notes about the trailer
  5. Convert notes to a working Java application or game

Option 2: IMDB Trailers

  1. Go to IMDB’s trending trailers: https://www.imdb.com/trailers/
  2. Search Google for “random number between 1 and 6” and click Generate
  3. The number returned from number 2, is the number trailer to watch (count from left to right)
  4. Make notes about the trailer
  5. Convert notes to a working Java application or game
